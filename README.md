# MFA Ambassadors Website

A responsive static website for MFA Ambassadors — the grassroots movement supporting Rt. Hon. Mojeed Fatai Adebola in Ibeju-Lekki Constituency I, Ward E.

## Files

- `index.html` — homepage with About Us, History, Projects (including CEC table), Mission & Vision, and contact form.
- `styles.css` — responsive styling with dark theme.
- `script.js` — form submission behavior.
- `images/` — directory for team photos (currently using placeholder URLs).

## Open locally

1. Open `index.html` in a browser.
2. Or use Live Server in VS Code for automatic refresh.
